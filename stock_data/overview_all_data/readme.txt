overview_all_data文件字段说明：
【股票代码】 股票的代码，上证股票以sh开头，深证股票以sz开头
【股票名称】 股票在本交易日的名称
【交易日期】 交易日期
【新浪行业】 股票所在的行业，一个股票可能属于多个行业
【新浪概念】 股票所在的概念、版块，一个股票可能属于多个概念
【新浪地域】 股票所在的地域，一个股票可能属于多个地域
【开盘价】 股票当天的开盘价
【最高价】 股票当天的最高价
【最低价】 股票当天的最低价
【收盘价】 股票当天的收盘价
【后复权价】 后复权价，复权开始时间为股票上市日，精确到小数点后10位
【前复权价】 前复权价，复权开始时间为股票最近一个交易日，精确到小数点后10位
【涨跌幅】 股票当天的复权之后的真实涨跌幅，保证准确
【成交量】 股票当天的成交量，单位(股)
【成交额】 股票当天的成交额，单位(元)
【换手率】 股票当天的换手率，成交量 / 流通股本
【流通市值】 流通市值，流通市值 / 股价 = 流通股股本
【总市值】 总市值，总市值 / 股价 = 总股本
【是否涨停】 当天收盘是否涨停，1代表是，0代表否
【是否跌停】 当天收盘是否跌停，1代表是，0代表否
【市盈率TTM】 最近12个月市盈率，股价 / 最近12个月归属母公司的每股收益TTM
【市销率TTM】 最近12个月市销率， 股价 / 最近12个月每股营业收入
【市现率TTM】 最近12个月市现率， 股价 / 最近12个月每股经营现金流
【市净率】 市净率，股价 / 最近期财报每股净资产
【MA_5】 5日移动平均线，所有均线数据根据前复权价格计算出
【MA_10】 10日移动平均线
【MA_20】 20日移动平均线
【MA_30】 30日移动平均线
【MA_60】 60日移动平均线
【MA金叉死叉】 显示当日移动平均线组合的金叉或者死叉信息，例如"5_10金叉"代表当日5日均线和10日均线金叉
【MACD_DIF】 MACD指标中DIF的值
【MACD_DEA】 MACD指标中DEA的值
【MACD_MACD】 MACD指标中MACD的值
【MACD_金叉死叉】 股票当天MACD指标是否金叉或死叉，DIF上穿DEA为金叉，下穿为死叉，分别为买入、卖出信号

【KDJ_D】 KDJ指标中D的值
【KDJ_J】 KDJ指标中J的值
【KDJ_金叉死叉】 股票当天KDJ指标是否金叉或死叉，K上穿D为金叉，下穿为死叉，分别为买入、卖出信号
【布林线中轨】 布林线指标中轨的值，计算公式与通达信行情软件一致。所有布林线指标中的数据根据前复权价格计算出。

【布林线下轨】 布林线指标下轨的值
